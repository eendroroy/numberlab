# numseries

Implementation of popular mathematical number series algorithms.

## Installation

To use the `numseries` library in your Rust project, add the following to your `Cargo.toml`:

```toml
[dependencies]
numseries = { git = "https://github.com/eendroroy/numseries" }
```
